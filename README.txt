Website Romantis (Bucin) — Cara pakai
1. Ekstrak zip ke folder di komputer Anda.
2. Buka file index.html dengan browser (double-click).
3. Isi nama pada halaman pertama lalu lanjutkan.
4. Semua data disimpan di localStorage pada browser — tidak perlu server.

Nama file:
- index.html  (Halaman 1 — Isi Nama)
- page2.html  (Halaman 2 — Pertanyaan Romantis)
- page3.html  (Halaman 3 — Kata-Kata & Puisi)
- styles.css  (Gaya / CSS)
- script.js   (Logika dan localStorage)

Catatan:
- Aplikasi bekerja langsung dengan membuka file HTML (no server).
- Google Fonts digunakan; koneksi internet diperlukan untuk memuat font. Jika ingin offline, Anda bisa mengunduh font Pacifico & Poppins dan menyertakannya secara lokal.
